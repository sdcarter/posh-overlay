# Quickstart

- `npm run storybook`: Launch the Storybook environment to visually test and iterate on the 0px border-radius styling and M-Sport colors using mock telemetry sweeps before dropping it directly into the running game instance.
- `npm run mock:mazda` (or any mock): Test the universal nature of the RPM mappings against various standard generic profiles to ensure they properly display the new RPM blocks.
