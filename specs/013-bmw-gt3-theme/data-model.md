# Data Model: bmw-gt3-theme

## Domain Layer Interactions (`src/domain/`)

- `TelemetrySnapshot`: We will confirm or inject the `ShiftIndicatorPct` into the snapshot generated by the iRacing adapter so the domain layer can use it universally.
- `RevStripState`: Will transition from a variable array of distinct LED configurations built via `latest-car-data` into a statically typed object or fixed array representing the fundamental M-Sport RPM states (Light Blue threshold, Dark Blue threshold, Red threshold, and Redline flash).
